# Student Management System
## Project Description
The Student Management System is a Python console application used to manage student records. The application supports CRUD operations and stores data using JSON file handling.
## Features Implemented
1. Add Student Record
2. View Student Records
3. Search Student Record
4. Update Student Record
5. Delete Student Record
6. JSON File Storage
7. Menu-Driven Interface
8. Object-Oriented Programming (OOP)
## Steps Performed
1. Created a Student Management System using Python.
2. Implemented CRUD operations.
3. Added file handling using JSON.
4. Tested all functionalities.
## Technologies Used
- Python
- JSON
- Object-Oriented Programming
