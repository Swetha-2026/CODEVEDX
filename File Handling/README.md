# File Handling & Data Processing System
##  Project Description
This Python project demonstrates file handling operations such as reading, writing, counting lines, counting words, and searching for words in a text file.
##  Features
- View file content
- Count total lines
- Count total words
- Search for a word
- Generate analysis report
- Exception handling (FileNotFoundError, General Exception)
- Menu-driven system
## Files Used
- data.txt → Input file
- file_processing.py → Python program
- Report.txt → Output report file
