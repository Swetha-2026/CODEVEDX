Student Management System

Project Overview

Student Management System is a Python-based console application developed as part of the CodeVedX Python Programming Internship.

The system helps manage student records efficiently by allowing users to add, view, search, update, and delete student information. Data is stored in a JSON file for persistence.

Features

- Add new student records
- View all student records
- Search students by ID
- Update student information
- Delete student records
- Store data in JSON format
- Menu-driven console application
- Input validation and error handling

Technologies Used

- Python 3
- JSON File Handling

How the System Works

Add Student

The user enters:

- Student ID
- Student Name
- Department
- Marks

The information is saved to the JSON file.

View Students

Displays all student records stored in the system.

Search Student

Allows users to search for a student using their Student ID.

Update Student

Users can modify existing student details.

Delete Student

Users can remove student records from the system.

How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the command:

python student_management.py

4. Select an option from the menu.
